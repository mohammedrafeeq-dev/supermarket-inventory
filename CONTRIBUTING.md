# Contributing to Supermarket Inventory

Thanks for your interest in contributing! We welcome improvements and bug fixes.

## Ways to contribute

1. Fork the repository
2. Create a feature branch: `git checkout -b feature-name`
3. Commit your changes: `git commit -m "Add feature X"`
4. Push to your branch: `git push origin feature-name`
5. Open a pull request to `main`

## Code style

- Use consistent naming for variables and functions
- Keep functions short and responsibilities clear
- Add comments for non-obvious logic

## Testing

- Make sure your changes compile and run
- Add tests if applicable
- Update the README if behavior changes

## Reporting issues

- Use GitHub Issues to report bugs and request features
- Include a clear description and reproduction steps
